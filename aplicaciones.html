Aplicaciones Web

Frontend

El frontend se refiere a la parte visual con la que el usuario interactúa directamente.

Tecnologías Frontend

  • HTML
  • CSS
  • JavaScript

Backend

El backend es la parte que maneja la lógica, las bases de datos y el servidor.

Tecnologías Backend

  • Base de datos
  • Servidor web
  • Lenguaje de programación más utilizados

FullStack

Un desarrollador FullStack trabaja tanto en el frontend como en el backend de una aplicación.
